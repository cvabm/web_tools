应用程序

音频

  • AIMP3 - 32 位音频处理和多格式播放。
  • Foobar2000 - Windows 版的免费音频播放器，支持各种音频格式，并且有很多很酷的特色功能。
  • Winamp - 能播放 MP3，MP2，WAV，VOCAls 和 MIDI 文件的音乐播放器。

视频

  • mpv - 媒体播放器。 (https://github.com/mpv-player/mpv)
  • PotPlayer - 多媒体播放器，具有广泛的编解码器集合，它还为用户提供大量配置选项。
  • VLC - 多媒体播放器和框架，用于播放 DVD，音频 CD，VCD 和各种流协议。 (https://github.com/videolan/vlc)

录屏

  • 土豆录屏（好用） - 蓝奏云
  • ScreenToGif - 它允许你录制屏幕的一部分区域并保存为 gif 或视频
  • screen2exe 录屏生成 exe 文件 (阿里云盘)
  • 在线录屏 https://recordscreen.io/
  • ocam、班迪录屏（这两个都挺好用） 阿里云盘

聊天客户端

  • Discord - Discord 是免费的语音和文本聊天客户端，适合玩家和非玩家。您可以从浏览器使用它，它可以在 iOS，Android，Windows，Mac 和 Linux 上使用。
  • Telegram - 一个专注于速度和安全性的消息应用程序，它速度极快，操作简单且免费。(https://telegram.org/apps)

压缩

  • 7-Zip - 用于处理压缩包的开源 Windows 实用程序。完美支持 7z，ZIP，GZIP，BZIP2 和 TAR 的全部特性，其他格式也可解压缩。 (http://www.7-zip.org/download.html)
