提醒：开始上传后，不要修改正在上传的文件，否则容易报错。

列出桶内文件并删除
val objectListing = s3.listObjects(BUCKET_NAME)
val objectSummaries = objectListing.objectSummaries
objectSummaries.forEach { summary ->
    LogUtils.e(summary.key)
    s3.deleteObject(BUCKET_NAME, summary.key)
}

删除桶
s3.deleteBucket(BUCKET_NAME)
  • v4 签名创建的桶，不能用 v3 签名去删。
  • 用 AWS3SignerType 创建的桶，删除时不要继续带这个签名设置。
  • 2.21.0 同时包含 S3 和 S4，默认走 S4。
  • S3SignerType 是 v2 签名，URL 有效期最长可到 2 年。
  • AWSS3V4SignerType 是 v4 签名，URL 有效期最长 7 天。
  • 北京区域：cn-north-1；宁夏区域：cn-northwest-1。

Windows `mc` 客户端

https://dl.min.io/client/mc/release/windows-amd64/mc.exe

连接与查看

mc config host add minio http://xxxx:9500 username password
mc config host list
mc ls minio
mc ls minio/capture-screen

创建与删除

mc mb minio/bucket
mc rb minio/bucket
mc rb minio/bucket --force
mc rm minio/bucket/hosts
mc rm minio/bucket/etc --recursive --force

上传与下载

mc cp E:\log.txt minio/bucket
mc cp minio/bucket/log.txt E:\log
mc cp minio/bucket/log.txt E:\log111.txt
mc cp /myfolder minio/bucket --recursive

匿名权限

mc anonymous get minio/bucket
mc anonymous set public minio/bucket
  • 常用命令重点记 `ls`、`cp`、`mb`、`rb`、`rm`、`mirror`、`config`。
  • 完整文档：https://min.io/docs/minio/linux/reference/minio-mc.html
