📡 HTTP 请求 · 即用
GET 请求（带模拟 Token + 表格展示）
jsonplaceholder 公开 API，控制台表格查看
fetch 
const url = 'https://jsonplaceholder.typicode.com/posts/1';
const token = 'Bearer demo-token';

fetch(url, { headers: { 'Authorization': token } })
  .then(r => r.json())
  .then(data => {
    console.log('📦 响应数据:', data);
    console.table(data);   // 对象/数组更直观
  })
  .catch(console.error);
✅ 公开测试接口，无需真实Token。console.table 可清晰展示字段。
POST JSON (带 Authorization 头)
发送结构化数据，模拟业务请求
fetch 
const url = 'https://jsonplaceholder.typicode.com/posts';
const payload = {
  title: '调试任务',
  body: '来自控制台的测试',
  userId: 42
};

fetch(url, {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json',
    'Authorization': 'Bearer fake-jwt-token'
  },
  body: JSON.stringify(payload)
})
  .then(r => r.json())
  .then(res => console.log('✅ 创建成功', res))
  .catch(console.error);
返回模拟的创建结果，包含 id。用于测试 POST 流程。
携带 Cookie 的请求 (同源/跨域凭证)
模拟登录态请求，自动带 cookie
include 
// 在需要登录态的页面运行，携带当前cookie
fetch('https://reqres.in/api/users/2', {
  credentials: 'include'   // 同源/跨域均发送凭证
})
  .then(r => r.json())
  .then(user => console.log('👤 用户信息', user))
  .catch(console.error);
reqres.in 为公开测试API。若需携带真实cookie，可将URL换成同域地址测试。
⏱️ 轮询请求 (带停止句柄)
每 N 秒拉取数据，可随时终止
定时 
const pollUrl = 'https://jsonplaceholder.typicode.com/posts/1';
const intervalMs = 3000;

const pollTimer = setInterval(async () => {
  try {
    const res = await fetch(pollUrl);
    const data = await res.json();
    console.log(`🔄 [${new Date().toLocaleTimeString()}]`, data);
  } catch (e) { console.warn('轮询出错', e); }
}, intervalMs);

// 停止轮询: 执行 clearInterval(pollTimer)
console.log('⏹️ 停止轮询: clearInterval(pollTimer)');
轮询句柄保存在 pollTimer，需要停止时执行 clearInterval(pollTimer)。
🔌 WebSocket · 测试回声
WebSocket 连接 (wss echo，支持token参数)
公开回声服务，发送即返回
WS 
const token = 'demo-token-123';
const ws = new WebSocket(`wss://echo.websocket.org?token=${token}`);

ws.onopen = () => {
  console.log('✅ WebSocket 已连接 (携带token参数)');
  ws.send(JSON.stringify({ type: 'ping', msg: 'hello echo', ts: Date.now() }));
};

ws.onmessage = (e) => console.log('📨 回声:', e.data);
ws.onerror = (e) => console.error('❌ 错误', e);
ws.onclose = () => console.log('🔌 连接关闭');

// 关闭: ws.close()
echo.websocket.org 稳定可靠。Token 通过query传递 (浏览器WebSocket无法自定义header)。
WebSocket 连续发送 (测试吞吐)
快速发送多条消息，观察回声
WS 
const ws2 = new WebSocket('wss://echo.websocket.org');
ws2.onopen = () => {
  [1, 2, 3].forEach(i => {
    ws2.send(JSON.stringify({ seq: i, payload: `msg-${i}` }));
  });
  console.log('📤 已连续发送3条消息');
};
ws2.onmessage = (e) => {
  try {
    const data = JSON.parse(e.data);
    console.log(`⬅️ 收到 seq=${data.seq}`, data);
  } catch { console.log('收到:', e.data); }
};
回声服务会原样返回，可验证消息顺序。
🛠️ 调试 & 拦截
🔍 拦截所有 fetch 请求 (调试神器)
打印请求URL、参数及响应预览
拦截 
const originalFetch = window.fetch;
window.fetch = async (...args) => {
  const [url, options] = args;
  console.groupCollapsed(`🌐 fetch: ${url}`);
  console.log('options', options || {});
  const response = await originalFetch(...args);
  const cloned = response.clone();
  cloned.text().then(body => {
    try {
      const json = JSON.parse(body);
      console.log('📋 响应 (json):', json);
    } catch { console.log('📄 响应 (text):', body.slice(0, 200)); }
  }).catch(()=>{});
  console.groupEnd();
  return response;
};
console.log('✅ fetch 拦截已激活 (刷新页面后失效)');
所有fetch请求都会被记录。用groupCollapsed使日志整洁。刷新恢复。
📦 快速查看 localStorage / sessionStorage
一键打印当前域下存储内容
存储 
// 打印 localStorage (键值对)
console.log('📁 localStorage:');
for (let i = 0; i < localStorage.length; i++) {
  const key = localStorage.key(i);
  console.log(`  ${key}`, localStorage.getItem(key));
}

// sessionStorage
console.log('📁 sessionStorage:');
for (let i = 0; i < sessionStorage.length; i++) {
  const key = sessionStorage.key(i);
  console.log(`  ${key}`, sessionStorage.getItem(key));
}
快速排查本地存储，无需打开Application面板。
🍪 查看当前页面 Cookie
解析 document.cookie 为对象
cookie 
const cookies = document.cookie.split(';').reduce((acc, pair) => {
  const [key, val] = pair.trim().split('=');
  if (key) acc[key] = decodeURIComponent(val || '');
  return acc;
}, {});
console.log('🍪 Cookies:', cookies);
// 也可用 console.table(cookies)
console.table(cookies);
仅能访问非HttpOnly的cookie。控制台快速查看。
📋 复制 fetch 响应为全局变量
将请求结果存为 temp1，方便操作
实用 
fetch('https://jsonplaceholder.typicode.com/users/1')
  .then(r => r.json())
  .then(data => {
    window.temp1 = data;
    console.log('✅ 已存为 temp1，可直接使用', data);
  });
之后在控制台输入 temp1 即可访问数据，方便进一步调试。
🎨 常用 console 快捷指令
table, group, time, assert 等
备忘 
// 性能计时
console.time('操作'); /* 中间代码 */ console.timeEnd('操作');

// 表格展示数组/对象
console.table([{a:1,b:2}, {a:3,b:4}]);

// 分组折叠
console.group('详情'); console.log('...'); console.groupEnd();

// 断言
console.assert(1===2, '不会打印');

// 计数
console.count('事件');
直接复制对应行执行，提升调试效率。
